# Instalação

## Projetos
Para instalar os projetos não existe ordem. Basta seguir os comandos. 

### VSS-Vision
```
git clone https://github.com/
cd VSS-Vision
sudo ./configure.sh
```

### VSS-State
```
git clone https://github.com/linesousa/VSS-Time-B-Robotica-Movel_2019/tree/master/VSS-State
cd VSS-State
sudo ./configure.sh
```

### VSS-Command
```
git clone https://github.com/
cd VSS-Command
sudo ./configure.sh
```
